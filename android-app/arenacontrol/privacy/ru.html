Термины и определения


  • ArenaControl - Android-приложение
  • MyArena - хостинг игровых серверов, расположенный по адресу https://myarena.ru
  • Токен - Строка, выданная клиенту MyArena в разделе API
  • ScCraft (ScCraftRU) - Чиварзин Александр Евгеньевич

Политика конфиденциальности


ArenaControl передаёт данные через официальный API MyArena с использованием протокола HTTPS.
ArenaControl может логировать токен с использованием утилиты android.util.Log
С целью персонализации рекламы и учёта её статистики ArenaControl использует ваш рекламный интендификатор.
Пользователь принимает эту политику автоматически при добавлении любого сервера в ArenaControl, и/или при интендификатор согласии на использование рекламного интендификатора
ArenaControl никуда не передаёт ваш рекламный интендификатор
ScCraft оставляет за собой право изменять эту политику без каких либо уведомлений. Изменения вступают в силу, после публикации новой версии политики на сайт!
